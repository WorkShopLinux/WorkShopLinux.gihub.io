#Plugins
A seguir estão listados os plugins que faço uso atualmente, para instalação de qualquer um deles basta ter instalado o package control e por meio do atalho *Ctrl + Shift + P* buscar por *“Package Control: Install Package”*, esperar que a lista de plug-ins seja carregada e buscar pelo plugin.

- Anaconda – Ótimo plugin fornece todo o suporte para trabalhar com projetos em python.
- CSS Format – Útil para manipulação de arquivo CSS.
- FolderFiles – Permite adição de pastas na sidebar.
- SideBarEnhacements – Libera opções para manipulação de arquivos e pasta com o clique direito na sidebar.
- Markdown Preview – Fornece recursos para pré visualizar arquivos feitos com markdown.

#Temas
A instalação dos temas é semelhante ao dos plug-ins, mas é possível realizar alguns ajustes de acordo com a sua preferencias por meio da settings – user. Os plug-ins para o tema que uso atualmente são: Flatland, Monokai Extended, Afterglow.
